# Vantage - SE3092 Personal Finance Management App

This is an Android application developed for SE3092 Platform Based Development Assignment 01.

The application is designed to help users track multi-source income, expenses, recurring payments, and savings goals. The app addresses the financial challenges faced by ambitious people with multiple income sources.

## Technology Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Firebase Authentication
- Firebase Firestore
